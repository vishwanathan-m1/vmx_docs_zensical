Upgrading Volt MX Foundry
============================

This section explains upgrading Volt MX Foundry Components through the installer.

> **_Important:_** Before you upgrade Foundry, refer to the [Certified and Supported Configurations](https://help.hcl-software.com/voltmx/v9.5/Foundry/voltmxfoundry_supported_devices_os_browsers/Content/Introduction.html), and make sure that you upgrade the underlying software stack.

*   [Upgrading Volt MX Foundry to 9.5 or later](https://help.hcl-software.com/voltmx/v9.5/Foundry/voltmx_foundry_linux_install_guide/Content/Upgrading_VoltMX_Foundry_SP1.html) : You can upgrade to Volt MX Foundry V10 by providing the existing Foundry database details.

*   [Upgrading the Custom Metrics](https://help.hcl-software.com/voltmx/v9.5/Foundry/voltmx_foundry_linux_install_guide/Content/CustomMetrics.html)
